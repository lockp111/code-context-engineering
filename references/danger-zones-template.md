# ⚠️ 危险区域模板

修改以下区域需要特别谨慎。

---

## 高风险文件

| 文件/目录        | 风险原因   | 修改建议                |
| ---------------- | ---------- | ----------------------- |
| `core/auth.ts`   | 认证核心   | 需要完整测试 + 安全审查 |
| `migrations/`    | 数据库迁移 | 只能追加，禁止修改已有  |
| `config/prod.ts` | 生产配置   | 需要人工审核            |
| `shared/utils/`  | 高被依赖   | 修改需全量回归          |

---

## 敏感操作清单

### ❌ 禁止操作

- 不要直接操作生产数据库
- 不要硬编码密钥或凭证
- 不要删除或重命名公开 API
- 不要修改已有的数据库迁移文件

### ⚠️ 需要审批

- 修改数据模型需要迁移计划
- 更新依赖版本需要兼容性测试
- 修改认证/授权逻辑
- 修改第三方集成

---

## 安全注意事项

- 所有用户输入必须验证
- 敏感数据必须加密存储
- API 必须有适当的权限检查
- 日志不能包含敏感信息
