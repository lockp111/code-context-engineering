# 常见任务路径模板

Agent 完成常见开发任务的标准流程。

---

## 添加新 API 端点

1. 在 `src/routes/` 创建路由文件
2. 在 `src/services/` 添加业务逻辑
3. 在 `src/types/` 定义请求/响应类型
4. 在 `tests/api/` 添加集成测试

---

## 添加新 UI 组件

1. 在 `src/components/` 创建组件目录
2. 创建 `index.tsx`, `styles.css`, `types.ts`
3. 在 `tests/components/` 添加单元测试

---

## 修复 Bug

1. 先写失败的测试用例
2. 定位问题代码
3. 修复并确保测试通过
4. 检查相关模块是否受影响
